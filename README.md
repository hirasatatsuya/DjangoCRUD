# DjangoCRUD

**確認手順**

１. ローカル環境内にクローンさせる

```git clone https://github.com/hirasatatsuya/DjangoCRUD.git```

2. クローンしたらcdコマンドで対象のディレクトリまで移動

```cd DjangoCRUD/ToDoApp/mydiaryproject```

３. mydiaryproject直下でサーバーの立ち上げ

``` python manage.py runserver ```

4. ブラウザ上でTOP画面のURL指定

``` http://127.0.0.1.8000/index ```
